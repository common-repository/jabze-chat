== Installation ==

Install Jabze Chat for Wordpress in 3 simple steps:
 
1. Download & Install the latest Jabze Chat for Wordpress plug-in
2. Activate the plug-in on the configuration page
3. Sync your users to our secure Jabze cloud platform
 
Optional: 

* Download our Jabze apps for [iPhone](https://itunes.apple.com/us/app/jabze/id685991823?mt=8&uo=4) 
	or [Android](https://play.google.com/store/apps/details?id=com.jabze.android)
* Access our service with our feature-rich web app at [www.jabze.com](https://www.jabze.com/)
 
(NOTE: Click the (?) on your Wordpress plugin and press the iPhone or Android button to retrieve your user name & password thatâ€™s required to login to these clients)
